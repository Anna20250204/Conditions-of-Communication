Between Meanings

Linguistic incompleteness across cultures

Click image to enter website ↗

Description

This project explores incompleteness between languages through a website centered on proverbs from different cultures. Rather than treating translation as a complete transfer of meaning, the project highlights the differences that appear in tone, phrasing, and cultural context.

By comparing similar sayings across cultures, the work suggests that meaning is never fully fixed. It is shaped by language, interpretation, and the conditions through which it is encountered.
